# bonfleur-website
Websiteentwicklungsprojekt für So26

Ziel:   -eine website für den familien blumenladen zu erstellen
        -bestand website erstellen
        -warenkorb (temporärer speicher)
        -bestellung (speicher) auich stonierbar
         


Die seite soll es den kunden ermöglichen blumen auf der website zu bestellen.
der kunde sieht welche ware Verfügbar ist (und die Anzahl der ferfügbarkeit) 
sowie ist in der lage seine bestellung im warenkorb temporär zu speichern und
nach einer bestellung in seinen account diese bestellung einsehen. der account
soll aes auch ermöglichen die bestellung zu stonieren.
